# Finera User Manual

Osnovni kostur projekta za korisnicko uputstvo uradjen pomocu **Material for MkDocs**.

## Pokretanje lokalno

1. Instalirati zavisnosti:

```bash
pip install -r requirements.txt
```

2. Pokrenuti lokalni server:

```bash
mkdocs serve
```

3. Otvoriti adresu:

```text
http://127.0.0.1:8000
```
